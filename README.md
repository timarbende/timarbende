# Hi there, I'm Bende 👋

## 🚀 About Me

I am a software developer and researcher focused on the intersection of **Computer Vision**, **Generative AI**, and **Graphics**. I enjoy building tools that bridge the gap between 2D data and 3D environments.

- 🔭 **Currently working on:** Room-scale PBR material generation using Score Distillation Sampling (SDS).
- 🧠 **Interests:** Neural Radiance Fields (NeRFs), Physically Based Rendering (PBR), and Diffusion Models.
- 🌱 **Learning:** Advanced CUDA programming and specialized 3D generative architectures.
- 💬 **Ask me about:** Texture generation pipelines and indoor scene reconstruction.
